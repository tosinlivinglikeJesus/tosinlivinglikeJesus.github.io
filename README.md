# BellaBeat Fitness Case Study

## Overview
Analysis of FitBit smart device data to uncover insights 
into user behaviour and inform BellaBeat's marketing strategy.

## Tools Used
- R (tidyverse, ggplot2, lubridate, janitor).
- RStudio.
- R Markdown.

## Data Source
FitBit Fitness Tracker Data — Kaggle (CC0 Public Domain)  
Published by Möbius. 30 FitBit users, April–May 2016.

## Limitations
- Small sample size (30 users).
- Data collected in 2016 — user behaviour may have changed.
- Weight dataset had only 8 unique participants (24% coverage) 
  and was excluded from analysis.

## Key Findings
- Users are most active at 6:00 PM.
- Saturday is the most active day of the week.
- Higher sedentary time correlates with less sleep.
